## Featured Projects

### [Pennywise](https://github.com/Nupoor-K/pennywise) — iOS Expense Splitter
> SwiftUI · Core Data · MVVM · MessageUI

Privacy-first trip expense splitter. Calculates net balances and generates the minimum set of settlement transfers (at most N−1 for N participants) using a greedy creditor/debtor algorithm. Entirely on-device — no account, no cloud sync.

- Programmatic Core Data schema with normalized entities for trips, expenses, and settlements
- Self-correcting balance model: settlements recorded as flagged expenses, no special-casing needed
- In-app email reminders via `MessageUI`
- 30+ unit tests covering balance calculation, rounding edge cases, and settlement correctness

---

### [MovieDB](https://github.com/Nupoor-K/MovieDB) — iOS Movie Discovery App
> SwiftUI · TMDB API · Swift Concurrency · NSCache · MVVM

Movie search and discovery app with zero third-party dependencies. Features debounced search, weekly trending, and a persistent favourites list.

- Actor-wrapped `NSCache` (250 images / 80 MB) with in-flight request deduplication
- `TaskGroup`-based image prefetch for the featured carousel
- Protocol-based networking layer enabling full unit test coverage without network access
- Atomic JSON writes for favourites; disk errors surface as user-facing alerts
